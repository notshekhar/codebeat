What is Node?

Node.js is a JavaScript interpreter for writting server side applications. In its simplest form it follows you to trigger small JavaScript programs from the command line without any browser involved. For Example: To run JavaScript program in a file called hello.js 


        console.log("Hello World!");

And then use terminal with "node hello.js" 

        
          $ node hello.js
          Hello World!



Node Package Manager

Node package manager (npm) comes with node and allows you to install JavaScript packages.
Node Packages are like JavaScript Libraries and you can work in the same way as JavaScript Library. So, to get start first you need to create a node project using "npm install" 

        
          $ npm init
